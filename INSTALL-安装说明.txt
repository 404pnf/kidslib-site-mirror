
# windows服务器安装
1. 复制shaoerguan到服务器任意硬盘根目录下
2. 在桌面创建mongoose5.exe的快捷方式，命名为“启动外研社少儿馆”
3. 双击mongoose5.exe启动程序
4. 点击windows工具条右下角托盘的黄色小动物标志，点go to my address； 或者打开浏览器访问 http:/127.0.0.1:8080
5. 打开网站后，在桌面创建一个到网站的快捷方式，命名为“访问外研社少儿馆”

# linux版本
1. 复制shaoerguan到服务器任意目录下
2. 在服务器kidslib目录下启动服务 ./mongoose5-linux
3. 打开浏览器访问 http:/127.0.0.1:8080
4. 打开网站后，在桌面创建一个到网站的快捷方式，命名为“访问外研社少儿馆”
5. 可以编辑 /etc/rc.local加入 /安装目录/mongoose5-linux 这样开紧急自动启动

# mac osx 版本
1. 复制shaoerguan到服务器任意目录下
2. 在服务器kidslib目录下启动服务 ./mogoose5-macosx
3. 打开浏览器访问 http:/127.0.0.1:8080

5. 按时不知道如何开机直接启动
